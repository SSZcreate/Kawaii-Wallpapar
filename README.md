# Kawaii-Wallpapar

LIVELY WALLPAPERで壁紙に設定する用のwebpageです。  
ブラウザ上で3Dキャラクター（VRMモデル）と触れ合える、インタラクティブなWebビューワーです。  
マウス操作による視線追従、クリックによるリアクション、自動まばたきなどの機能を備えています。

## ✨ 特徴

### 生きているような表現

- **視線追従**: マウスカーソルの位置をキャラクターが目で追います。
- **自動まばたき**: ランダムなタイミングで自然なまばたきを行います。
- **プロシージャルアニメーション**: 待機中や手を振る動作などをプログラムで制御しています。

### インタラクション

- クリックでキャラクターが喜びます（ジャンプ＆表情変化）。
- 放置すると時々こちらに手を振ってくれます。

### VRMファイル読み込み

- ドラッグ＆ドロップで、手持ちのVRMモデルに切り替えることができます。

### 高品質なレンダリング

- 3点照明（キー、フィル、リムライト）によるスタジオ風ライティング。
- ACES Filmic トーンマッピングによる色鮮やかな描画。

## 📦 Lively Wallpaper 設定手順

このHTMLファイルをLively Wallpaperで動かすための手順です。  
VRMモデル（AliciaSolid.vrm）を使用する前提で説明します。

### 準備

#### フォルダの作成

デスクトップなど、好きな場所に新しいフォルダを作成してください。（例: `VRM_Wallpaper`）

#### ファイルの配置

作成したフォルダの中に、以下の2つのファイルを入れます。

- `index.html` (先ほど作成したコード)
- `AliciaSolid.vrm` (ダウンロードしたモデルファイル)

> **注意**: ファイル名が違う場合は `index.html` 内の `modelPath` を書き換えるか、ファイル名を `AliciaSolid.vrm` に変更してください。

### Lively Wallpaperへの登録

以下のどちらかの方法で登録してください。

#### 方法A: ドラッグ＆ドロップ（簡単）

1. Lively Wallpaperのウィンドウを開きます。
2. 画面右上の+ボタンから壁紙の追加を開きます
3. URLを入力に以下をコピペしてください
https://sszcreate.github.io/Kawaii-Wallpapar/
4. OKボタンを押します
5. 追加された壁紙を自分のデスクトップに設定します。

### うまく動かないとき

#### マウスに反応しない

- Livelyの設定で「Mouse Input（マウス入力）」が有効になっているか確認してください。
- 壁紙設定の「Interaction」タブを確認してください。

## 🚀 ローカル環境での使い方

このプロジェクトはWebブラウザで動作しますが、ローカルファイル（VRM）を読み込むセキュリティ制約（CORS）のため、ローカルサーバーを立てて実行することを推奨します。

### 1. セットアップ

プロジェクトのディレクトリに移動し、簡易サーバーを立ち上げます。

**Python 3系の場合:**

```bash
python -m http.server
```

**VS Codeの場合:**

拡張機能「Live Server」をインストールし、`index.html` を右クリックして "Open with Live Server" を選択してください。

### 2. ブラウザで開く

ブラウザで `http://localhost:8000` （または指定されたポート）にアクセスします。

### 3. 操作方法

- **マウス移動**: 視線誘導
- **左クリック**: ジャンプ（リアクション）
- **ドラッグ＆ドロップ**: 任意の `.vrm` ファイルを画面にドロップするとモデルが切り替わります。
